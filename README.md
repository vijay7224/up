🤖 IntelliChat AI – OpenAI Chatbot

IntelliChat AI is a smart conversational chatbot built using the OpenAI API.
It can answer user queries, generate responses, and simulate human-like conversations through an interactive web interface.

🚀 Features

💬 Real-time chat interface
🤖 AI-powered responses using OpenAI
⚡ Fast and intelligent replies
🌐 Web-based UI
🧠 Context-aware conversation

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Flask (Python)
AI Model: OpenAI API
Libraries: openai, flask
